cern-rpmverify

Verifies the integrity of the files which are installed by rpm packages

Download SLC 5 source rpm Download SLC 6 source rpm

View on GitHub

Description

cern-rpmverify is designed to detect changes in the files that are installed by rpm packages. It will log the changed files as well as the rpm packages that they belong at the local syslog facility. Configuration files are ignored, because they are supposed to be changed. You need root priviledges to execute cern-rpmverify, otherwise all the files that need root privileges will be ignored.

IMPORTANT

If you want to run a more strict rpm verification, execute 'rpm -V --all'. With this command you will get information about not only changed files, but changes in permissions etc.

Format of the log messages


Feb  9 14:50:23 localhost python: cern-rpmverify: started
Feb  9 15:04:02 localhost python: cern-rpmverify: File /usr/bin/catchsegv from package glibc-common-2.12-1.47.el6 has been modified
Feb  9 15:04:04 localhost python: cern-rpmverify: File /usr/bin/consolehelper from package usermode-1.102-3.el6 has been modified
Feb  9 14:55:15 localhost python: cern-rpmverify: finished
